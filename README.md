# MGILT Next

MGILT 麦吉蓝图官网的新开发主线。项目从旧版静态网站整理而来，旧仓库保留为历史档案，本仓库用于后续内容、视觉和功能优化。

## 本地运行

项目目前是原生 HTML、CSS 和 JavaScript 网站，不需要安装依赖或执行构建。

```bash
python3 -m http.server 8806 --bind 127.0.0.1
```

浏览器访问 `http://127.0.0.1:8806/`。

不能直接双击 `index.html` 进行完整测试，因为 3D 编辑器使用 ES Module，需要通过本地 HTTP 服务加载。

## 目录

- `index.html`：网站页面、样式和主要交互。
- `css/product-system.css`：产品系统页面的独立样式。
- `js/product-system.js`：产品系统页面的详情与订购交互。
- `js/editor-3d.js`：Three.js 空间设计器。
- `js/gsap.min.js`、`js/ScrollTrigger.min.js`：页面动画依赖。
- `images/`：网站当前实际使用的图片。
- `models/`：3D 编辑器当前已有的 GLB 模型。

## 当前已知问题

- 登录和注册仍是旧版前端模拟实现，不可作为正式账号系统。
- 合作申请、留言等表单尚未接入正式后端。
- 编辑器代码引用了学校白板和三名学生模型，但旧仓库未包含对应 GLB 文件。
- `index.html` 仍是大型单文件，后续将逐步拆分，同时保持页面行为一致。

## 安全约定

- 本仓库不连接当前 `mgilt.com` 正式服务器。
- 所有发布先进入独立预览环境，确认后再决定是否绑定新域名。
- 大范围修改前保留可回退提交。
